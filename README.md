# Detecting Masquerade Attacks in Controller Area Networks Using Machine Learning

## 📖 Overview

This project focuses on detecting masquerade attacks in Controller Area Networks (CAN) using Machine Learning. It analyzes CAN bus messages, constructs graph-based relationships using NetworkX, and classifies messages as normal or malicious using the Random Forest algorithm. A user-friendly Tkinter GUI is included to simplify prediction and visualization.

---

## 🎯 Objectives

- Detect masquerade attacks in CAN networks.
- Improve vehicle network security using Machine Learning.
- Build graph-based features from CAN messages.
- Provide an easy-to-use graphical user interface.

---

## 🛠️ Technologies Used

- Scikit-learn

---

## 🔍 Machine Learning Algorithm

**Random Forest Classifier**

The model is trained using CAN dataset features and predicts whether a CAN message is normal or an attack.
